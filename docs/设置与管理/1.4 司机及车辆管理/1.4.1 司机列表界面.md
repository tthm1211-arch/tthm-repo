**司机列表**页面路径： 管理端 → 配置中心 → 司机管理 → 司机列表

### 功能整合计划
当前版本中存在三个与司机信息相关的页面：
- 配置中心 → 司机管理 → 司机列表
- 配置中心 → 操作管理 → 数据管理 → 人员管理 → 司机管理 → 司机信息（正向）
- 配置中心 → 操作管理 → 数据管理 → 人员管理 → 退件司机
后续版本计划将上述三个页面的全部字段及相关信息统一合并至司机列表中。

### 司机类型介绍
目前系统中的司机类型主要分为以下三种：

| 司机类型   | 业务模块        | 说明            |
| ------ | ----------- | ------------- |
| 正向派送司机 | 正向派送模块      | 负责正向派送和正向揽收业务 |
| 逆向退件司机 | 退件揽收模块      | 负责退件揽收业务      |
| 正逆向司机  | 正向派送 + 退件揽收 | 同时承担正向和逆向业务   |

司机列表中显示的包括全部三种类型的司机。计划在版本更新后，将司机类型字段调整为：**正向 / 逆向 / 正逆向**，按司机所属业务模块进行分类。
### 司机列表界面
<ZoomImage src="/Pasted%20image%2020260616162629.png"/>
司机列表界面分为搜索区、操作区和数据列表区三个区域。

#### 搜索区
	搜索规则：
	单一条件搜索：数据列表显示所有包含该关键词的结果
	    示例：在"司机号"搜索框中输入"10"，将显示所有司机号包含"10"的司机
	复合条件搜索： 数据列表仅显示符合全部搜索条件的结果
	若需清除某个搜索条件，可删除搜索框内容或点击【重置】按钮

1. 搜索条件:
	- 司机号：管理端中每个司机的唯一标识符，司机与司机号为一对一关系。司机号在创建司机时自动生成，创建逻辑详见[司机列表维护](./1.4.2%20司机列表维护.md)
	- 司机姓名：支持模糊搜索，搜索时将忽略大小写差异
	- 手机号码：搜索时存在以下限制条件——无法识别含有中划线（-）的号码，号码前缀也不支持"1"或"+1"的格式
	- 启用状态：单选项，可选值为启用或停用
	- 司机类型：单选项，当前可选值为中大件、卡派、干线。后续版本更新时将调整为"正向 / 逆向 / 正逆向"，按司机所属业务模块进行分类
	- 工作方式：单选项，可选值为外包、T4A、T4
	- 审核状态：单选项，可选值为待审核、已驳回、已通过
2. 搜索按钮：点击触发搜索动作
3. 重置按钮：点击清除所有已选择项，重置筛选条件

#### 操作区
 4. 新增按钮：新增司机记录，司机号系统自动生成，其他信息需手动填写。详细字段说明见《司机列表维护 - 新增司机》
 5. 导出按钮：默认情况下导出司机数据列表中的全部数据，有筛选条件时导出所有符合条件的数据。目前司机列表界面不支持自定义导出字段，且导出文件的字段排序与界面显示顺序不一致，需手动调整
#### 数据列表区
6.  表头字段：包含以下信息：

	| 序号  | 字段名     | 说明                |
	| --- | ------- | ----------------- |
	| 1   | 序号      | 行号                |
	| 2   | 司机号     | 司机唯一标识            |
	| 3   | 司机类型    | 正向 / 逆向 / 正逆向     |
	| 4   | 司机姓名    | 司机实名              |
	| 5   | 工作方式    | 外包 / T4A / T4     |
	| 6   | 手机号码    | 联系电话              |
	| 7   | 准入时间    | 审核通过时间            |
	| 8   | 注册时间    | 初次创建时间            |
	| 9   | 驾照类型    | 驾照等级              |
	| 10  | 驾照有效期   | 驾照失效日期            |
	| 11  | 计费模式    | T4为时薪制；T4A/外包为计件制 |
	| 12  | 所属承运商   | 关联承运商             |
	| 13  | 紧急联系人   | 紧急联系人姓名           |
	| 14  | 紧急联系人电话 | 紧急联系电话            |
	| 15  | 工作状态    | 启用 / 停用           |
	| 16  | 启用状态    | 当前状态              |
	| 17  | 审核状态    | 待审核 / 已驳回 / 已通过   |
	
	
7. 操作列：包含以下操作功能：
	- 查看：跳转进入【司机详情】界面，该页面中所包含的字段和信息将在[司机列表维护](./1.4.2%20司机列表维护.md)--新增司机 中进行详细介绍。仅查看，无编辑权限
	- 修改：跳转进入【司机详情】界面编辑信息，司机号不可修改，其他字段可编辑
	- 删除：删除对应司机记录，删除操作不可逆
	- 审核：审核司机信息，仅对审核状态为"待审核"的司机有效，可更新为"已通过"或"已驳
