### 司机号生成逻辑
新建司机时，系统根据所选大区自动生成对应前缀的司机号。不同大区生成的司机号互不重复。若未能匹配到大区映射关系，系统将提示异常并禁止生成默认编号。司机号前两位与大区的对应关系如下：

| 大区  | 司机号前两位 |
| --- | ------ |
| YYZ | 10     |
| YVR | 20     |
| YUL | 30     |
| YYC | 40     |
| YEG | 50     |
### 新增司机
1. 前往司机列表界面，路径为管理端 → 配置中心 → 司机管理 → 司机列表
2. 在操作区点击【新增】按钮，系统打开【司机详情】编辑页面
3. 填写所有必填字段。系统支持的字段类型如下：
	-  **文本字段**：司机姓名、手机号码、驾照号、邮编、邮箱、家庭地址、社保号、紧急联系人、紧急联系人电话
	- **单选项**：启用状态（建议选择"启用"）
	- **下拉菜单**：国家、省份、大区、驾照类型、工作方式、司机类型
	- **日期选择**：驾照有效期
	- **图片上传**：身份证正面、身份证反面、驾照正面、驾照反面
4.  点击【确定】按钮保存
5. 系统返回司机列表页面并显示【操作成功】提示，新增流程完成
### 审核司机
1. 在司机列表中定位目标司机 （可在搜索区中【司机号】或【司机姓名】搜索框中输入关键词），点击该司机所在行【操作】列的【审核】按钮
2. 系统打开【司机审核】弹窗。选择【通过】后，将触发【计费模式】下拉菜单出现
3. 根据司机的工作情况，从【计费模式】下拉菜单中选择以下其中一种：
	- 计件机制：按件数计费
	- 时薪机制：按小时计费
	- 时薪保底机制：按小时计费且有保底
	
	计费模式确定后，在【司机详情】页面中**无法修改**。若需调整计费模式，只能删除该司机后重新创建。
4. 点击【确定】按钮提交审核结果
5. 系统返回司机列表并显示【操作成功】提示。目标司机所在行的【准入时间】单元格将自动填充审核通过时间，审核流程完成

### 修改司机信息
1. 在司机列表中定位目标司机，点击该司机所在行【操作】列的【修改】按钮
2. 系统打开【修改】弹窗，显示该司机的现有信息。可编辑字段包括：司机姓名、手机号码、驾照号、邮编、邮箱等
3. 修改相关信息。以下两个字段创建后不可修改：
	- **司机号**：作为唯一标识符，无法更改
	- **计费模式**：若需调整，必须删除后重新创建
4. 点击【确定】按钮保存修改
5. 系统返回司机列表并显示【操作成功】提示，修改流程完成
### 删除司机
1. 在司机列表中定位目标司机，点击该司机所在行【操作】列的【删除】按钮
2. 系统弹出删除确认提示，内容为"此操作将永久删除该司机，删除后无法恢复"
3. 确认无误后，在确认弹窗中点击【确定】按钮
4. 系统返回司机列表并显示删除成功提示，删除流程完成
### 导出司机信息
1. 在搜索区中设置查询条件以筛选需要导出的司机数据；如不设置筛选条件则默认导出数据列表中的所有司机信息
2. 在操作区点击【导出】按钮
3. 系统生成Excel文件（xlsx格式），直接保存至本地，无需经过后台任务中心转存。导出的Excel文件采用固定的字段排列顺序，具体如下：
	| 序号  | 字段 |
	| --- | ------ |
	| 1. | 司机姓名 |
	| 2. | 司机类型 |
	| 3. | 工作方式 |
	| 4. | 手机号码 |
	| 5. | 准入时间 |
	| 6. | 注册时间 |
	| 7. | 驾照类型 |
	| 8. | 驾照有效期开始 |
	| 9. | 驾照有效期结束 |
	| 10. | 计费模式 |
	| 11. | 所属承运商 |
	| 12. | 紧急联系人 |
	| 13. | 紧急联系人电话 |
	| 14. | 工作状态 |
	| 15. | 启用状态 |
	| 16. | 审核状态 |

系统提供【表格视图切换】工具可调整页面显示中的字段顺序，但此工具的调整不会影响导出文件的字段排列。导出文件始终保持上述默认的固定字段顺序。若导出文件需按特定顺序排列，请在Excel软件中手动调整字段位置。关于【表格视图切换】工具的详细使用说明，请参考[仓库信息界面](../1.3%20仓库管理/1.3.1%20仓库信息界面.md)文档中的相关章节。

### 常见问题
1. 新创建的司机通过审核后为什么无法立即在分配司机或绑定车辆时选择？
	为提升系统性能，司机、批次等基础数据会缓存至 Redis，因此新增数据不会实时同步到查询列表。
	
	创建司机后，如暂时无法在分配司机或绑定车辆页面搜索到该司机，建议先刷新页面并重新查询；若仍未显示，请等待缓存同步完成后再进行操作。
