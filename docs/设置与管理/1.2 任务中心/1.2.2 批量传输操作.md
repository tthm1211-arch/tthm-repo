### 功能页面
支持批量下载的页面：
- 订单管理 → 中大件订单列表 → 导出
- 订单管理 → 快递订单列表 → 门店/代理 → 导出
- 调度管理 → 预约揽收 → 导出
- 订单管理 → 退件揽收单 → 右侧下载图标

支持批量上传的页面：
- 订单管理 → 中大件订单列表 → 导入
- 订单管理 → 机场提货列表 → 导入
- 订单管理 → 退件揽收单 → 导入
- 调度管理 → 预约揽收 → 批量创建

如上图所示，功能页面从上至下分为三个区域：
- 搜索区：设置筛选条件，支持搜索框输入和下拉选择

### 异步下载机制
管理端的批量下载采用异步机制，分为两个步骤：
1. 生成文件：在功能页面点击【导出】后，系统在后台生成文件并存入任务中心，不会自动下载至本地
2. 下载文件：可参考*2.2.1 任务中心界面 - 任务中心访问入口* 前往任务中心，手动点击【下载】按钮，将已生成的文件保存至本地

### 批量下载操作
#### 设置筛选条件
导出前必须设置筛选条件，否则点击【导出】时页面将提示「导出必须选择时间范围」。
系统端支持以下两种筛选方式：

	1. 搜索框筛选
	在功能页面上方搜索区（可点击展开或收起）设置筛选条件：在对应字段旁的输入框中输入关键词，或在下拉选择器中选择筛选值。如需定位某字段，可使用 Ctrl+F 进行页面内查找，匹配的字段名称将高亮显示。系统端支持同时设置多个筛选条件，设置完成后点击【确定】如需清除筛选条件，可逐个删除、点击【重置】或刷新页面批量清除。		
		
	上述三个页面（中大件订单列表、快递订单列表和预约揽收列表）均支持通过跟踪单号或客户单号进行批量查询，多个单号之间以换行或逗号分隔。

	2. 复选框勾选
	在数据列表左侧复选框列勾选目标数据：点击表头复选框可全选当前页数据。每页展示条数可调节，最少 1 条，最多 100 条，默认 50 条。翻页后前一页已勾选的数据将自动清除，请在翻页前完成导出操作

**两种方式的优先级说明：**

当搜索框筛选条件与复选框勾选数据同时存在时，系统将以勾选数据为准进行导出，并弹出系统提示弹窗告知用户。如下图所示，因此建议根据实际需求单独使用其中一种方式，避免冲突。

#### 导出操作步骤
1. 前往目标功能页面（路由路径参见本节「功能页面」）
2. 在搜索区设置筛选条件，或在数据列表区左侧复选框列勾选目标数据
3. 点击操作区中的【导出】按钮
4. 页面弹出「前往任务中心」弹窗，点击【前往任务中心】
5. 在任务传输列表中根据创建时间找到对应任务，点击操作列中的【下载】按钮
6. 导出文件将以只读 .xlsx 格式在新标签页中打开，表头字段与功能页面保持一致，可另存至本地后进行编辑

#### 导入操作步骤
1. 前往目标功能页面（路由路径参见本节「功能页面」）
2. 点击操作区中的【导入】按钮
3. 在「导入数据」弹窗中上传 .xls 或 .xlsx 格式文件
4. 点击【确认】完成导入

### 中大件订单列表批量导出注意事项
1. 当前版本暂不支持自定义导出
	在当前版本（**v1.0.90**）中，【中大件订单列表】的自定义导出功能存在异常。使用该功能导出文件时，可能出现 "文件格式与文件扩展名不匹配" 的提示，导致文件无法正常打开。
	该问题为当前版本已知问题，建议暂时使用默认导出功能。待问题修复后，本知识库将同步更新相关说明。 
    <ZoomImage src="/Pasted image 20260716170716.png"/>
	
2. 导出文件数据脱敏
	为保障数据安全，当前版本（**v1.0.90**）导出的 Excel 文件已默认对敏感信息进行脱敏处理。
	以下字段将以脱敏后的内容进行导出：
	- 发件人姓名
	- 发件人电话
	- 发货详细地址
	- 收件人姓名
	- 收件人电话
	- 收件详细地址

3. 支持按订单或按包裹导出
	为满足**一票多件**业务场景，中大件订单列表支持以下两种导出方式：
	- 按订单导出：以订单为维度导出数据，每个订单仅生成一条记录
	- 按包裹导出：以包裹为维度导出数据，一票多件订单将按照实际包裹数量生成多条记录
