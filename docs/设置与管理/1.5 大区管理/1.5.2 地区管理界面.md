**地区管理页面**路径：管理端 → 配置中心 → 中大件数据 → 大区管理 → 区域管理

### 地区概念解释
地区（区域）是大区的细分管理单位。一个大区可划分为多个地区，一个地区对应一个二级派送仓，并包含多条路线和若干邮编范围。
<ZoomImage src="/Pasted%20image%2020260618125632.png"/>

**与其他模块的关系**：
- **仓库层级**：一个大区对应一个一级分拣仓，一个地区对应一个二级派送仓，一级分拣仓下有多个二级派送仓
- **路线层级**：一个地区可被划分为多个路线，每条路线对应特定的配送轨迹
- **邮编范围**：地区通过FSA（邮编前三位）定义其服务覆盖范围
- **派送流程**：司机的派送任务根据路线进行路径规划，派送前往地区对应的二级仓进行取件，然后按路线进行配送
### 搜索区

<ZoomImage src="/Pasted%20image%2020260618132415.png"/>

支持三个搜索条件，均支持模糊搜索：
- 地区名称：命名规则因大区而异

| 大区          | 命名规则        | 示例               |
| ----------- | ----------- | ---------------- |
| YVR（温哥华）    | 大区编码 + 区域位置 | YVR-North Surrey |
| YYZ（多伦多）    | 二级仓名称-路线编号  | 99301-VG10-T1X26 |
| YUL、YYC、YEG | 地理方位名       | Edmonton-South   |

- 路线编号：该地区所属的路线编号
- 覆盖邮编：输入邮编前三位（FSA）进行搜索

### 操作区
4.  新增按钮：创建新地区需指定以下信息：二级仓、启用状态、路线编号、覆盖邮编。一般情况下无需新增地区。
### 数据列表
5. 表头字段：包含以下信息
	- 序号：列序号
	- 区域名称：命名规则见搜索区说明，字段右侧有地图图标，点击弹出Google Maps窗口，展示该区域的覆盖范围和相对位置
	- 仓库名称：该地区对应的二级派送仓库名称；YYC、YUL、YEG大区中地区名称与仓库名称一致
	- 仓库类型：本页面中所有仓库均为二级仓
	- 路线编号：该地区所属的路线编号
	- 覆盖邮编：该地区覆盖范围内的所有FSA三字邮编，鼠标悬停显示详细邮编
	- 覆盖数量：该地区覆盖的邮编总数
	- 启用状态：显示地区是否启用/停用
	- 操作：支持编辑、停用或删除该地区
