分拣作业在各口岸对应的分拣仓库中进行。分拣仓库配备充足的作业人员，支持自动分拣和人工分拣两种模式，可执行货物上架、分拣、数据核对等操作。

仓库基础信息详见 [1.3.1 仓库信息界面](/docs/设置与管理/1.3%20仓库管理/1.3.1%20仓库信息界面.md)。
#### 分拣流程 -- 库内
```
货物入库
  ↓
─→ [分拣机分拣成功？]
   ├─ YES → 自动挂批次 → 系统自动归批 → 完成
   └─ NO  → 手动挂批次 → 人工分拣 → 数据核对 → 完成
```

**自动分拣流程**
1. 批次号创建：对应口岸分拣机每日开机时自动创建该口岸当日的批次号
2. 货物分拣：货物经由分拣机挂上对应批次号
3. 数据同步：管理端自动将货物归入该批次，完成系统同步

**人工分拣流程**

当分拣机分拣失败或无法处理时，转换为人工分拣模式：
1. 人工分拣：分拣人员在仓储app中手动为待扫描包裹关联当日的批次号
2. 数据录入：在管理端录入相关信息

#### 分拣流程 -- 管理端
分拣在管理端中涉及的模块位置如下：

```
├─ 管理端 
│ ├─ 仓库管理
|  │ ├─ 分拣机管理
|  │ ├─ 人工分拣管理
│ ├─ 调度管理
│ └─ 批次管理
```

| 模块     | 路径            | 功能                |
| ------ | ------------- | ----------------- |
| 分拣机管理  | 仓库管理 → 分拣机管理  | 管理自动分拣机的运行状态、参数配置 |
| 人工分拣管理 | 仓库管理 → 人工分拣管理 | 处理手动分拣、异常分拣等操作    |
| 批次管理   | 管理端 → 批次管理    | 管理分拣批次号、批次信息查询    |
