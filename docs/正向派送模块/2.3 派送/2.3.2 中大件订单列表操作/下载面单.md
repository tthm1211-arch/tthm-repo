使用场景：当原面单损坏、条码无法识别，导致司机无法正常扫码取件时，可在管理端重新下载对应订单的面单，用于重新打印或辅助司机完成扫码取件。

### 下载单个订单面单
功能路径：订单管理 → 中大件订单列表
#### 操作步骤
1.  在【中大件订单列表】中查询目标订单（可参考[3.3.1 中大件订单列表界面](/docs/正向派送模块/2.3%20派送/2.3.1%20中大件订单列表界面.md)）
2.  在该订单的【操作】列点击【更多】
3.  在下拉菜单中选择【下载面单】
4.  系统将直接下载该订单的面单文件

<ZoomImage src="/Pasted image 20260716175547.png" alt="下载面单" />

#### 文件说明
-   文件格式：PDF
-   下载方式：直接下载至本地，不经过【任务中心】
-   文件名称：默认使用该订单号命名

### 批量下载订单面单
功能路径：订单管理 → 中大件订单列表

### 操作步骤
1.  在数据列表中勾选需要下载面单的订单
2.  点击功能操作区中的【下载面单】
3.  系统将批量生成并下载所选订单的面单

<ZoomImage src="/Pasted image 20260716180220.png" alt="下载面单" />

#### 文件说明
-   文件格式：PDF
-   下载方式：直接下载至本地，不经过【任务中心】
-   文件命名规则：`labels_下载日期 下载时间`
-   示例：`labels_2026-07-16 174317`

### 注意事项
1. 批量下载必须先勾选订单
	仅在搜索区域筛选订单后直接点击【下载面单】，系统会提示："请选择行数据"
	因此，批量下载面单必须通过勾选数据列表中的订单触发，不能仅依靠搜索条件执行下载
2. 单次最多下载 50 个订单
	批量下载面单的单次选择上限为 **50** 个订单。若一次勾选超过 50 条数据，系统将提示"单次选择数量上限为 50 个"

### 常见问题
若单个下载及批量下载均失败，并出现【无法下载面单】提示，建议按以下步骤排查：
1.  刷新页面后重新下载；
2.  确认订单是否存在有效面单数据；
3.  若问题仍未解决，请联系开发人员进一步排查
