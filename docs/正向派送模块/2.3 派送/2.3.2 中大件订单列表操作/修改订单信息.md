【中大件订单列表】页面中，订单可修改的内容会随着订单状态变化而有所不同。订单越接近完成，可修改的内容越少。

| 订单状态       | 支持修改的订单信息                |
| ---------- | ------------------------ |
| 待揽收、待运输    | 全局修改（除客户单号外，所有订单信息均可修改）  |
| 待运输后至订单完成前 | 仅支持修改收件人信息（地址相关）         |
| 已完成        | 不支持修改订单信息，仅支持修改派送证明（POD） |

### 全局修改
**适用状态**：
- 待揽收
- 待运输
仅以上两种状态支持全局修改，请先确认订单状态。

**操作路径**：订单管理 → 中大件订单列表 → 更多 → 修改
进入订单详情后，可修改除**客户单号**外的所有订单信息，包括运输信息、收发件人信息及货物信息。

### 修改地址
**适用状态**：
- 待运输
- 运输中
- 待收货
- 待返仓
- 已返仓
- 再次派送
- 派送失败
- 仓库收件
- 已代扫

**操作路径**：中大件订单列表 → 更多 → 修改 → 基本信息
可修改以下收件人信息：
-   联系人
-   联系电话
-   邮编
-   详细地址1
-   详细地址2
-   收件人经纬度

> **注意**
>
> 在【基本信息】页面中，仅【留言提醒】支持点击**编辑**进行修改，其他收件人信息仅可在支持修改地址的订单状态下编辑。

### 修改派送证明
**适用状态**：已完成

**操作路径**：中大件订单列表 → 更多 → 编辑证明

进入【编辑证明】弹窗后：
1.  确认需要修改的跟踪单号。
2.  鼠标悬停在已上传图片上，可查看或删除图片。
3.  点击 **+** 上传新的派送证明图片。
